# codeEditor
Build a code editor to compile and run your code
The project uses moncao-editor package to perform as an online editor and third-party apis to compile the programs written.
Clone the repository in your machine and run the following after navigating to it:


              $ npm install      (to install dependencies)
              $ npm start        (to run the application)
